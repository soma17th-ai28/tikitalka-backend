## Codebase Patterns
- Project is Python FastAPI at the soccer/ root (main.py)
- Pydantic v2 style validators: use @field_validator with @classmethod
- mypy is the available typecheck tool (pyright not installed)
- Run checks: `python -m mypy main.py --ignore-missing-imports`
- Start server: `python -m uvicorn main:app --host 0.0.0.0 --port 8000`
- tikitalka-backend/ is an empty Spring Boot skeleton — all new code goes in soccer/ root
- Upstage API endpoint: https://api.upstage.ai/v1/chat/completions, model: solar-pro (OpenAI-compatible)
- LLM response parsing: structured REPLY:/SUGGEST: format in system prompt; parse_llm_response() extracts both fields
- End-signal detection: check if any END_SIGNALS keyword appears in lowercased user message
- httpx.AsyncClient used for async LLM calls; raise HTTPException(502) on httpx errors
