## Codebase Patterns
- Project is Python FastAPI at the soccer/ root (main.py)
- Pydantic v2 style validators: use @field_validator with @classmethod
- mypy is the available typecheck tool (pyright not installed)
- Run checks: `python -m mypy main.py --ignore-missing-imports`
- Start server: `python -m uvicorn main:app --host 0.0.0.0 --port 8000`
- tikitalka-backend/ is an empty Spring Boot skeleton — all new code goes in soccer/ root
